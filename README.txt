Descargar fotos de Instagram:

Esta es una app creada en java, que permite la descarga de imagenes de instagram a nuestra computadora, a traves de urls de las imagenes de los perfiles de instagram, estas urls tienen el siguiene patrón: https://www.instagram.com/x/xxxXxXXxX.

Una vez visualizada la imagen en la app, podremos hacer click en ella y elegir en donde queremos guardar la imagen, le asignamos un nombre y una extensión (nombre_imagen.jpg, nombre_imagen.png, etc) y listo!.

Esta app funciona para cualquier Sistema Operativo (Windows, Linux, Mac).*

* Para poder ejecutar esta aplicación es necesario tener instalado el JDK (Java Development Kit) correspondiente a nuestro Sistema Operativo.
